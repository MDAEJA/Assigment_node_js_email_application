# Node.js Email Sending Application

This Node.js application uses Express and Nodemailer to send emails through a form. It sets up a basic server, provides an HTML form for user input, and sends the email using Gmail's SMTP settings.

## Features

- **Express Server**: Serves a simple web form and handles form submissions.
- **Nodemailer Integration**: Configured to send emails using Gmail's SMTP service.
- **HTML Form**: Collects email, subject, and message from the user.
- **Styled Email**: Sends a well-formatted email with basic CSS styles.

## Prerequisites

- [Node.js](https://nodejs.org/)
- [NPM](https://www.npmjs.com/)
